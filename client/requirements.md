## Packages
lucide-react | Icons for the interface
react-hook-form | Form state management
@hookform/resolvers | Zod validation integration for forms
zod | Schema validation
framer-motion | Smooth micro-interactions and list animations
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge tailwind classes without style conflicts

## Notes
- The application requires an endpoint `/api/playlist.m3u` to be available for downloading the generated playlist.
- Fonts used: 'Geist' for sans-serif and 'Geist Mono' for monospace elements to give a premium clean minimal aesthetic.
